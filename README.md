# 🎬 リール動画エディタ

スクリプトと画像から、Instagramリール動画（MP4）を自動生成するブラウザ完結型のエディタです。
テロップ（字幕）の自動分割や表示秒数の調整、画像へのモーション付与などが可能です。

## 🌟 特徴
- **ブラウザ完結**: サーバーへデータをアップロードせず、プライバシーを保護。
- **MP4 / SRT 出力**: Instagramへそのまま投稿できるMP4、CapCut等で使えるSRT字幕の両方に対応。
- **無音カット**: 音声ファイルから自動で無音区間を検出し、テンポの良い動画を作成。
- **手動編集**: 波形をマウスでドラッグして、言い間違えなどをピンポイントでカット可能。
- **モーションエフェクト**: 画像に「ズーム」「パン」などの動きを追加してリッチな演出。

## 🚀 使い方
1. **音声＆テキスト**: 音声ファイルをアップロードし、スクリプトを入力して「分割」ボタンを押します。
2. **テロップ編集**: 各テロップの文字や秒数を個別に調整します。無音カットもここで行えます。
3. **画像設定**: 各テロップに表示する画像を選び、モーション効果を選択します。
4. **プレビュー＆出力**: 仕上がりを確認し、「MP4でダウンロード」を押して完成です。

## ⚙️ 技術仕様
- **HTML/CSS/JS**: Vanilla JSによる高速な動作。
- **Web Audio API**: 音声解析と波形表示。
- **Canvas / WebCodecs API**: ブラウザ上での動画レンダリングとエンコード。
- **mp4-muxer**: 高速なMP4コンテナ生成。
